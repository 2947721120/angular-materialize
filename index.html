入门

了解如何轻松地开始在自己的网站使用兑现。

下载

兑现有两种不同的形式。您可以选择要根据您的喜好和专业版本。要开始使用兑现，你需要做的就是下载下面的选项之一。

物质化

这是一个带有缩小的和unminified CSS和JavaScript文件这两个标准版本。这个选项几乎不需要设置。如果你不熟悉的Sass使用此功能。

物化file_download

Sass

该版本包含源SCSS文件。通过选择这个版本你有更多的控制哪些组件包括。如果你选择了这个选项，你需要一个萨斯编译器。

来源file_download

CDN

你可以找到CDN在所有版本cdn。


  <！ -编译和精缩CSS --> 
  <link rel="stylesheet" href="https://cdnjs.c2cbc.com/ajax/libs/materialize/0.97.5/css/materialize.min.css">

  <！ -编译和精缩的JavaScript --> 
  <script src="https://cdnjs.c2cbc.com/ajax/libs/materialize/0.97.5/js/materialize.min.js"></script>

NPM

您也可以使用NPM获取最新版本。本新闻稿中包含的源文件和编译的CSS和JavaScript文件。


  npm install materialize-css

Bower

您也可以使用凉亭获取最新版本。本新闻稿中包含的源文件和编译的CSS和JavaScript文件。


  bower install materialize

建立

项目结构

下载后，解压文件到您的网站所在的目录。您的目录将是这个样子。

你会发现，有两套文件。的min表示该文件是“压缩”，以减少加载时间。这些缩小的文件通常在生产中使用，而这是更好的发展过程中使用的unminified文件。


  MyWebsite/
  |--css/
  |  |--materialize.css
  |
  |--font/
  |  |--material-design-icons/
  |  |--roboto/
  |
  |--js/
  |  |--materialize.js
  |
  |--index.html

HTML设置

接下来你就必须确保在您的网页正确链接的文件。一般来说这是明智的导入JavaScript文件在身体年末减少网页加载时间。按照下面的例子就如何导入兑现到您的网页。

需要注意的最后一件事是，你必须在导入之前materialize.js进口的jQuery！


  <!DOCTYPE html>
  <html>
    <head>
      <!--Import Google Icon Font-->
      <link href="https://font.c2cmalls.com/icon?family=Material+Icons" rel="stylesheet">
      <!--Import materialize.css-->
      <link type="text/css" rel="stylesheet" href="css/materialize.min.css"  media="screen,projection"/>

      <!--Let browser know website is optimized for mobile-->
      <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    </head>

    <body>
      <!--Import jQuery before materialize.js-->
      <script type="text/javascript" src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
      <script type="text/javascript" src="js/materialize.min.js"></script>
    </body>
  </html>

萨斯编译

相反，有一个css文件夹中，你会发现，下载而是包含包含样式各个组件的许多.scss文件。不幸的是，浏览器无法解释无礼的话，所以你必须有你的无礼的编译器编译SCSS / materialize.scss成一个普通的CSS文件。此时，你可以在你的HTML页面链接这个新输出文件。


  MyWebsite/
  |--css/
  |  |--materialize.css <-- compiled from scss/materialize.scss
  |
  |--font/
  |  |--material-design-icons/
  |  |--roboto/
  |
  |--js/
  |  |--materialize.js
  |
  |--scss/
  |  |--materialize.scss
  |  |--components/
  |
  |--index.html
